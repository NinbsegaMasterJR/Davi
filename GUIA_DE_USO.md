# Guia de Uso - Scriptura

Bem-vindo. Este guia mostra como usar o Scriptura para preparar pregacoes, estudos e materiais biblicos.

## Acesso

- Site: `https://scriptura-web.vercel.app`
- API: `https://scriptura-web-api.vercel.app`

## O que voce pode fazer

- Gerar esboco de pregacao
- Buscar versiculos por tema
- Fazer analise teologica
- Explicar uma passagem
- Consultar concordancia biblica
- Montar cronograma de pregacoes
- Criar carta pastoral

## Fluxo Basico

1. Entre no site publicado.
2. Escolha a ferramenta desejada.
3. Preencha tema, passagem ou parametros.
4. Gere o conteudo.
5. Copie, baixe ou refine no workspace.

## Exemplos

### Esboco

- Tema: `Fe`
- Estilo: `Pentecostal`
- Duracao: `40 minutos`

### Analise

- Tema: `Santificacao`
- Passagem: `1 Tessalonicenses 4:3`
- Profundidade: `medio`

### Versiculos

- Tema: `Cura`
- Limite: `10`

## Dicas

- Temas mais especificos costumam gerar respostas melhores.
- Revise sempre o resultado antes de usar publicamente.
- Salve materiais importantes no workspace.

## Duvidas Frequentes

**Precisa de internet?**  
Sim. O app depende da API para gerar conteudo.

**Posso editar o resultado?**  
Sim. O Scriptura entrega uma base para voce adaptar.

**Qual URL devo usar?**  
Use `https://scriptura-web.vercel.app`.
